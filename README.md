# 坦克大战升级版

一个纯 `HTML + CSS + JavaScript` 的浏览器版坦克大战，无需构建，直接打开 `index.html` 即可运行。

## 模式

- `基地防守`：双人协作守住基地，清空敌军波次并推进三张地图
- `双人对战`：同屏 1v1，先完成 `5` 次击毁获胜

## 操作

- `P1`：`W A S D` 移动，`F` 发射
- `P2`：方向键移动，`Enter` 发射
- `R`：快速重开

## 升级内容

- 三张基地防守地图和两张对战地图
- 砖墙、钢墙、水面、草丛四类地形
- 随机补给：护盾、修复、急速装填
- 更强的敌方寻路和射击逻辑

## 运行

直接用浏览器打开 [index.html](./index.html)。
